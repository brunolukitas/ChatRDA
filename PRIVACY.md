# Política de Privacidade - [Nome do Seu Bot]

Esta Política de Privacidade descreve como o [ChatRDA] ("Bot") coleta, usa e protege as suas informações. Nós nos comprometemos a garantir a privacidade dos usuários e a transparência no tratamento de dados.

## 1. Dados que Coletamos
Para funcionar corretamente, o Bot pode coletar e processar as seguintes informações fornecidas publicamente pelo Discord:
* **IDs de Usuário, Servidor e Canais:** Números de identificação exclusivos gerados pelo Discord para processar comandos.
* **Mensagens e Conteúdo:** Apenas o conteúdo de comandos direcionados explicitamente ao Bot.
* **Dados de Configuração:** Configurações personalizadas que você define para o Bot no seu servidor.

## 2. Como Usamos os Dados
Os dados coletados são utilizados exclusivamente para:
* Executar as funções e comandos do Bot solicitados pelos usuários.
* Armazenar preferências e configurações específicas do servidor (ex: canais de texto de boas-vindas ou cargos).
* Manter a segurança, prevenir abusos e corrigir erros no código.

## 3. Compartilhamento de Dados
* **Não vendemos, trocamos ou transferimos** suas informações para terceiros.
* Os dados não são compartilhados com nenhuma entidade externa, exceto quando exigido por lei ou pelas autoridades do Discord.

## 4. Retenção e Exclusão de Dados
* Os dados são armazenados apenas pelo tempo necessário para cumprir suas funções.
* Se você remover o Bot do seu servidor, os dados de configuração locais poderão ser mantidos por até [30 dias] dias antes de serem apagados definitivamente.
* Para solicitar a exclusão imediata de todos os seus dados armazenados, entre em contato pelo nosso suporte.

## 5. Alterações nesta Política
* Esta política pode ser atualizada para refletir mudanças no Bot. Avisaremos sobre mudanças significativas no nosso servidor de suporte.

## 6. Contato
* Se tiver dúvidas sobre a privacidade dos seus dados, Entre no supporte offical do discord em: [https://support.discord.com/hc/].
